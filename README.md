# tp-sc
Slideshow Control Plugin for Touch Portal

Version: 0.9.0

Info: Plugin is meant to enable Touch Portal to communicate with Powerpoint.

Copyright 2020 by jsilvanus@gmail.com. License MPL 2.0.
